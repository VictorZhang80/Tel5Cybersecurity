Assessment Report Options

Network Full Detail Report:

The Full Detail Report provides a complete assessment and documentation of Active Directory and Workgroup environments. Every aspect of the network from servers, workstations, users, groups, printers, network shares, applications, patching, to Internet access and speed test are documented in this detailed report. View Sample Report (PDF)

Security Risk Report:

This Security Risk Report is similar to other Risk Reports such as the Network Risk Report except that it focuses on security related issues. It includes an overall Risk Score, an overall Issues Score, as well as a summary list of issues discovered. This Risk Report identifies specific risks that are due to misconfigurations and violations of best practices. View Sample Report (PDF)

Microsoft Cloud Risk Report:

This Risk Report is similar to other Risk Reports such as network, and security except it covers Azure AD and the key Microsoft 365 services. It includes an overall Risk Score, an overall Issues Score, and a list of discovered issues based on Microsoft controls and best practices. View Sample Report (PDF)

Consolidated Security Report:

Presents an intuitive computer-by-computer view including a grade for overall security health, antivirus, local firewalls, patches, and OS support. Quickly identifies problem machines. View Sample Report (PDF)

Executive Summary:

Summarizes criticality of issues across categories like network, hardware, software, passwords, Microsoft Cloud, and more. Includes Issue Score and potential data breach risk. View Sample Report (PDF)

User Security Awareness Report:

Provides insight into employee security posture—training engagement, phishing simulation results, risk scores. Helps identify high-risk users and improve awareness. View Sample Report (PDF)
